EliotOS

EliotOS

EliotOS is minimal Linux Operating System, built with Linuxkit, which contains only minimal components to run Eliot. At the moment, EliotOS supports only RaspberryPI3.

  • Linux kernel
  • containerd - Container runtime
  • runc - Run containers based on OCI specification
  • eliotd - Primary node agent which manages containers based on specs

There's more info in project GitHub page.

See installation for info how to easily install it to RaspberryPI!
